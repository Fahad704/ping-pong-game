To play the game Open Ping Pong.exe file!

-----------CONTROLS-----
:::MENU
Down to select the button below and up to select button above

:::GAMEPLAY:
up-down keys to move up and down
left key to move down and Right Key to move up
WASD to control left player in multiplayer Mode

ESC button to return to menu

:::SCORE
if the ball hits your x side of the arena the opponent gains a point and vise versa
